HTML-верстка
Создание каркаса сайта, основанного на готовом макете или макет накидывается в процессе создания, отталкиваясь от требований зааказчика. Использую в свое работе семантику, FlexBox и Grid
JavaScript
JS — это язык, который создает последовательности действий в зависимости от событий, то есть различные события на сайте могут приводить к разным сценариям действий и разным конечным результатам. Все веб-браузеры понимают JavaScript и я пользуюсь этим, чтобы веб-страницы делали сумасшедшие вещи!
Разработка приложений
В работе использую React-библиотека для создания пользовательских интерфейсов. Она позволяет создавать простые представления для каждого состояния приложения, а также эффективно обновлять и отображать только нужные компоненты при изменении данных.

Что я использую при разработке
HTML5
Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта
CSS3
Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!
Java Script
Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое
React
Эта библиотека позволяет создавать web-приложения. Мы можем создать максимально интерактивный продукт именно под ваши цели
Typescript
Язык программирования, представленный Microsoft в 2012 году и позиционируемый как средство разработки веб-приложений, расширяющее возможности JavaScript
Redux Toolkit
Пакет, который предоставляет разработчикам набор утилит для создания приложений Redux. Он предназначен для упрощения и оптимизации процесса разработки за счет сокращения шаблонного кода и предоставления готовой поддержки для распространенных случаев
